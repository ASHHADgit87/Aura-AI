# Aura AI

**Aura AI** is a unified, AI-powered full-stack web application designed to provide 8 sophisticated AI tools at zero cost. From image generation and PDF summarization to code explanation and web scraping, Aura AI delivers professional-grade functionality with an immersive and interactive interface. Built with MERN Stack, it combines a visually stunning frontend with high-performance AI models for professional utility.

---

## 🏛️ Architecture

**Aura AI follows a Multi-Tier Client–Server Architecture**, separating frontend, backend, and AI services for scalability, maintainability, and fast performance.

---

## ☁️ SaaS 

Aura AI follows the **Software as a Service (SaaS)** model by providing centrally hosted AI functionality that users can access online without any local installation.

---

## 🔐 Authentication & Security

Aura AI includes a **custom-built authentication system** designed to handle many real-world security scenarios commonly addressed in production applications.

Key implementations include:

- Custom **JWT-based authentication** with secure token generation  
- **Token expiration handling and automatic session logout**  
- Protected API routes using authentication middleware  
- Secure password hashing using **bcrypt**  
- Account existence tracking to improve authentication UX  

This authentication system was designed to simulate **industry-level security workflows**, addressing many common edge cases encountered in modern SaaS platforms.

## ✨ Features

### 🎨 Frontend
- Immersive 3D Bubble Background with Three.js and React Three Fiber  
- GlowCard interface for smooth card animations and tool presentation    
- Fully responsive for Desktop, Tablet, and Mobile  

### ⚙️ Backend & AI Logic
- Node.js and Express.js for RESTful API management  
- MongoDB for user and tool data storage  
- Advanced AI integration for image generation, PDF summarization, code explanation, background removal, translation, grammar fixing, and web scraping  
- Optimized data flow and minimal latency for real-time AI responses  
- Modular architecture for easy addition of new AI tools
